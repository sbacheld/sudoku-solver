# sudoku-solver
A Sudoku solver built using various CSP techniques.

## Environment Setup

Use Conda to create the environment.
```bash
conda env create -f env.yml
```

Activate the environment:
```bash
conda activate sudokusolver
```

Deactivate the environment:
```bash
conda deactivate
```

## Tests

Running all tests:
```bash
python -m unittest
```
